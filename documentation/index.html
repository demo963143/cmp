Laundry Management System

Introduction

Complete PHP script laundry management system. Multilanguage (English/arabic)
Using Codeigniter 3 ,ajax bootsrap 4

Installation

System Requrements

*** PHP 7.1

*** MYSQL 5+

*** PHP MYSQLI Extension

Installation

Download and unzip the package if you haven't already. Create a database for Laundry on your web server if it is possible. In case there is no MySQL user who has all privileges for accessing and modifying this database, create it as well. Upload the app files and folders to the chosen location on your web server.
Run the laundry app installation script by accessing the public URL for the chosen location in a web browser. This should be the URL where you uploaded the app files.
http://yourdomain.com/pathtoupload/install/
Follow the instructions on each screen of the installer. Once the installation is done, delete the /install folder

Manually install

  1. Unzip and copy the script folder in your hosting server.
  2. Make sure in script folder index.php and .htaccess file have permission as 644
  3. Make sure in script folder uploads and application/config folder have permission as 777
  4. Database file (DB/laundry.sql)
  5. Manually create a database using “Phpmyadmin”.
  6. Manually Database config application/config/database.php.
